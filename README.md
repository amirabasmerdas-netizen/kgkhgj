## 🚀 نصب محلی

```bash
# ۱. کلون کردن پروژه
git clone <your-repo-url>
cd amel_self55

# ۲. نصب وابستگی‌ها
pip install -r requirements.txt

# ۳. تنظیم متغیرهای محیطی
cp .env.example .env
# فایل .env را با مقادیر واقعی ویرایش کنید

# ۴. اجرا
python app.py
```

سپس به `http://localhost:5000` بروید و با شماره تلفن وارد شوید.

---

## ☁️ استقرار روی Render.com

1. **یک Web Service جدید بسازید**
2. **متغیرهای محیطی را تنظیم کنید:**
   - `API_ID`
   - `API_HASH`
   - `PHONE_NUMBER`
   - `SECRET_KEY`
3. **Build Command:** `pip install -r requirements.txt`
4. **Start Command:** `python app.py`
5. **پس از Deploy:** به آدرس سرویس بروید و login کنید

### UptimeRobot
برای جلوگیری از sleep، آدرس زیر را در UptimeRobot اضافه کنید:
```
https://your-app.onrender.com/keepalive
```
هر ۵ دقیقه یک‌بار ping می‌کند.

## 📌 دستورات کامل تلگرام

> ⚠️ **مهم:** دستورات باید **دقیقاً** همان‌طور که نوشته شده ارسال شوند. هیچ کلمه اضافه‌ای قبل یا بعد از دستور مجاز نیست.

### کنترل سلف
| دستور | توضیح |
|-------|-------|
| `سلف روشن` | فعال کردن سلف‌بات |
| `سلف خاموش` | غیرفعال کردن کامل |
| `وضعیت` | نمایش وضعیت همه سیستم‌ها |
| `راهنما` | نمایش راهنمای کامل |

### دوستان و دشمنان
| دستور | توضیح |
|-------|-------|
| `تنظیم دوست` | (ریپلای روی پیام) اضافه به لیست دوست |
| `حذف دوست` | (ریپلای) حذف از لیست دوست |
| `نمایش لیست دوست` | نمایش همه دوستان |
| `تنظیم دشمن` | (ریپلای) اضافه به لیست دشمن |
| `حذف دشمن` | (ریپلای) حذف از لیست دشمن |
| `نمایش لیست دشمن` | نمایش همه دشمنان |

### اتوماسیون
| دستور | توضیح |
|-------|-------|
| `منشی روشن` | فعال کردن پاسخ خودکار در پیوی |
| `منشی خاموش` | غیرفعال کردن منشی |
| `ضد حذف روشن` | ذخیره پیام‌های حذف‌شده |
| `ضد حذف خاموش` | غیرفعال کردن ضد حذف |
| `قفل پیوی روشن` | حذف همه پیام‌های پیوی |
| `قفل پیوی خاموش` | غیرفعال کردن قفل پیوی |
| `ضد لینک روشن` | حذف لینک‌ها در پیوی |
| `ضد لینک خاموش` | غیرفعال کردن ضد لینک |
| `دیدن خودکار روشن` | خواندن خودکار پیام‌ها |
| `واکنش خودکار روشن` | واکنش خودکار به پیام‌ها |

### مدیریت پیام
| دستور | توضیح |
|-------|-------|
| `ذخیره 1` تا `ذخیره 10` | ذخیره پیام ریپلای‌شده در اسلات |
| `ارسال 1` تا `ارسال 10` | ارسال پیام از اسلات |
| `حذف‌شده‌ها` | نمایش آخرین پیام‌های حذف‌شده |

### اسپم
| دستور | توضیح |
|-------|-------|
| `اسپم 5 سلام` | ارسال ۵ بار پیام "سلام" |
| `توقف اسپم` | توقف اسپم |
